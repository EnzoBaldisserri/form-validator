Form Field

A FormField is the representation or an HTML input fieldset controlled by the validator.

Outline

  {
    $el:             null,             // The HTML input field
    name:            null,             // The name of the field, if it's an a form
    type:            InputTypes.NONE,  // The type of the input
    validations:     [],               // The validations required for the input to be valid
    style: {
      validClass:    null,             // Class applied when the input is valid
      invalidClass:  null              // Class applied when the input is invalid
    },
    onInit:          null,             // Callback function called at the initialization of the field
    onChange:        null,             // Callback function called when the value of the field changes
  }

Details

$el

The HTML element.
Must be either an <input> or a <textarea>.

name

The name of the element.
This option is available only if the input field is in a form in the forms parameter.

Either $el or name must be set ! If both are, $el has priority over name.

type

The type of the field. (see input types)
If not given, the type will be taken from the type attribute if there is one, else it defaults to InputTypes.NONE.

validations

An array of validations. (see validations)
The validations represents the conditions for the field to be valid.

E.g.

  [
    Validations.NOT_EMPTY(),
    Validations.VALUE_IN({ min: 0, max: 50 }, 'my name'),
  ]

style

A Style object.
The style to be applied specifically to this element.
This style cannot be overrided by anything in the library.
If not set, it is herited from it's parent style (either Form style or FormValidator style).

onInit(value, properties)

Called when initializing the field. Returning false or setting onInit to false will prevent from applying validity classes to the element.

onChange(event, value, properties)

Called when the value of the field changes. Returning false will remove the validity classes from the element

The properties argument for both onInit and onChange is an object having this fields:

  • isValid: Whether the input is valid or not
  • valids: List of successfully passed validators
  • invalids: List of unsuccessfully passed validators
