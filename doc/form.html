Form

A Form is the representation of an HTML form from which some or all fields are controlled by the validator.

Outline

  {
    $el:               null,    // The HTML form element
    name:              null,    // The name, id, or index of the form in the document
    fields:            [],      // The fields to be controlled
    submit:            null,    // The submit button of the form
    style: {
      validClass: null,         // Class applied when the input is valid
      invalidClass: null,       // Class applied when the input is invalid
    },
    onValidityChange:  null,    // Callback function called when the valid state of the form changes
  }
