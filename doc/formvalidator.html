FormValidator

Outline

Initializing the validator is pretty simple, here's the outline:

  FormValidator.create({
    forms,  // Forms controlled by the FormValidator
    fields, // Fields that aren't in a form
    style,  // Style to be applied to every form and fields
  })

Details

forms

An array of Form objects.
If you only have one form to submit, you can omit the array.

fields

An array of Field objects.
Those fields shouldn't belong to a form. If they belong to one, you should prefer passing them throught the forms field.
If you only have one field to submit, you can omit the array.

style

A Style object.
This style can be overrided by forms and fields own style property.
